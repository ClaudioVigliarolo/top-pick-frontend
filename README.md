## Installation

1. Make sure you have the latest version of Node and React installed on your machine

1. Clone the repo
   ```sh
   git clone https://github.com/ClaudioVigliarolo/top-pick-frontend
   ```
1. Install NPM packages
   ```sh
   npm install
   ```
